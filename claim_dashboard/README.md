# Claim Resolution Report

## 📊 Report Objective
This report presents the **status of claim resolutions** by **branch, department, and employee**.  
It allows users to understand the **pending claims** as well as the **resolved claims**, supporting performance tracking and management decisions.

---

## 📁 Folder Structure
claim_dashboard/

├── dataset/ # Contains report datasets

├── REPORT_Claim_insurance (.pbix file)

---

## 📝 Report Contents

- **Claims by Branch**: Shows total pending and resolved claims for each branch.  
- **Claims by Department**: Evaluates the efficiency of claim handling per department.  
- **Claims by Employee**: Provides detailed insights into individual employee performance in resolving claims.  

---

## 🎯 Benefits

- Track claim resolution progress at multiple levels.  
- Identify bottlenecks and backlogs in the claims process.  
- Support management in making informed decisions and allocating resources effectively.  

---

## ⚙️ How to Use

1. Open the `.pbix` file in the `report/` folder.  
2. Connect to the dataset (update if new data is available) in the `dataset/` folder.  
3. Explore the interactive dashboards to monitor pending and resolved claims.
