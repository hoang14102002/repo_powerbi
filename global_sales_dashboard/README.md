# Global sales Report
## 📊 Report Objective

This dashboard provides a comprehensive overview of business performance across key domains, including finance, customer behavior, and operations.
It enables stakeholders to quickly assess the overall health of the business and make data-driven decisions.

---

## 📁 Folder Structure
claim_dashboard/

├── dataset/ # Contains report datasets and datadictionary file

├── REPORT_global_sales (.pbix file)

Another version of this dashboard I build by Looker Studio: [Link](https://datastudio.google.com/reporting/66755b52-7b11-4acd-a1c1-15cf6d64f6ea)

---

## 📝 Report Contents

- 📈 Finance Overview Provides insights into revenue, profit, and overall financial performance.
- 👥 Customer Behavior Analyzes customer trends, purchasing patterns, and segmentation.
- 🚚 Shipping & Returns Monitors delivery performance, shipping costs, and return rates to evaluate operational efficiency.

---

## 🎯 Benefits

- Gain a holistic view of business performance across multiple dimensions.
- Identify trends, patterns, and potential risks in finance and operations.
- Understand customer behavior to improve retention and sales strategies.
- Support strategic decision-making with interactive and visual insights.

---

## ⚙️ How to Use

1. Open the `.pbix` file in the `report/` folder.  
2. Connect to the dataset (update if new data is available) in the `dataset/` folder.  
3. Explore the interactive dashboards to monitor pending and resolved claims.
